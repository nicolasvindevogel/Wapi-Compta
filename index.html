Prochaine etape

WAPI Compta fonctionne maintenant autour d'un centre de traitement multi-coproprietes : import massif, reconnaissance automatique, correction humaine, validation puis creation des factures, extraits ou ecritures.

Centre de traitement multi-coproprietes

Import massif -> validation -> comptabilisation
Depose ici des factures PDF, CODA ou documents en masse. L'import reste separe de l'encodage financier : les CODA importes seront ensuite valides dans Comptabilite > Financier. Pour les factures, tu peux corriger la copro, le fournisseur, le montant et le compte comptable avant validation.
